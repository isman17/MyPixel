## Submission Capstone Akhir

### Submission checklist

- Menerapkan Continuous Integration:
	- Project CI MyPixel dapat diakses melalui link berikut: https://app.circleci.com/pipelines/github/isman17/MyPixel

- Menerapkan Security

	- Menerapkan obfuscation dengan ProGuard:
		- Teknik obfuscation dengan mengaktifkan proguard pada module app dan core
		- Untuk mengaktifkan proguard dengan cara memberikan nilai true pada minifyEnabled dan menambahkan konfigurasi pada consumer-rules.pro (module: core)

	- Menerapkan encryption pada database:
		- Menggunakan library SQLChiper dengan method openHelperFactory yang diterapkan pada database room tepatnya di module core/di/CoreModule.kt

	- Menerapkan certificate pinning untuk koneksi ke server:
		- Teknik certificate pinning dengan fungsi add menambahkan hostname server dan certificate server menggunakan CerticatePinning.Builder yang ditambahkan ke OkHttpClient.Builder dengan fungsi CertificatePinner
		- Certificate pinning diterapkan pada variabel network module tepatnya di core/di/CoreModule.kt
